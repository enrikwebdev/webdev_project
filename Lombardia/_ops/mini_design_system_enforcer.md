# Mini Design System + Enforcer (light)

Obiettivo: **consistenza premium** senza rallentare.

## Cosa è “sistema” (per noi)
Un set di pattern che **non si reinventa** a ogni sito:
- Hero (H1 chiaro + proof + CTA unica)
- Trust-strip (rating/anni/igiene/certificazioni)
- Servizi (cards + prezzi/da/a, se sensato)
- Proof (recensioni/foto reali)
- Contatti (mappa/telefono/orari) + CTA sticky mobile
- Footer (info legali minime)

## Enforcer (regola)
Prima di consegnare demo/outreach, fare un check “2 minuti”:
- CTA unica coerente (chiama/whatsapp/prenota)
- Proof vicino alla CTA (almeno 1)
- Mobile: sticky CTA presente
- Accessibilità base: focus visibile + contrasto ok
- Nessuna animazione pacchiana / eccessiva

## Quando è permesso deviare
Solo se motivato in 1 riga in `lead.txt` (sezione “Deviazioni dal sistema”) e se:
- migliora conversione
- non peggiora performance
- resta premium
